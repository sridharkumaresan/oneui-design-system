# Atom Golden Template

Use this folder as the copy source when creating a new atom.

Checklist for a new atom:

1. Copy template files into `src/components/<atom-name>/`.
2. Replace placeholder names and props.
3. Add unit tests and an axe accessibility test.
4. Add a colocated Storybook story.
5. Export from `src/components/index.ts` and `src/index.ts`.
